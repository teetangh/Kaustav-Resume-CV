# Prisma Flutter Connector

**Type:** Open-source library
**Timeline:** ~2025
**Repository:** `github.com/teetangh/prisma-flutter-connector`
**Language:** Dart

---

## Summary

A type-safe Dart client generator from Prisma schema, enabling Flutter apps to interface with Prisma-managed databases.

---

## Key Details

- Generates Dart client code from a Prisma schema definition
- Provides type-safe database access for Flutter applications
- Bridges the gap between Prisma ORM (typically Node.js) and the Flutter/Dart ecosystem

---

## Technologies

- Dart, Prisma ORM
- Code generation
- Flutter
