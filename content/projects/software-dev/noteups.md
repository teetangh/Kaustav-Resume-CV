## Summary

A marketplace for sharing and purchasing lecture notes. Users can upload, browse, and buy notes with integrated payment processing.

---

## Frontend

- **ReactJS** with **Redux Toolkit** and **RTK Query** for state management
- **Framer Motion** for animations
- **Styled Components** for component-level CSS
- **Sass** for CSS pre-processing
- **Media Queries** for responsive design
- **Font Awesome** for icons
- Deployed on **Netlify**

## Backend

- **Node.js** and **Express.js** for server-side rendering
- **MongoDB** as NoSQL database
- **Cloudinary** for PDF and metadata storage (images, note files)
- **PassportJS** for authentication:
  - JWT token-based auth
  - Social OAuth login
- Deployed on **Heroku**

## Key Features

- Synchronization of note products between Cloudinary and MongoDB
- **Stripe** integration for subscription payments
- Razorpay integration (was in progress)
