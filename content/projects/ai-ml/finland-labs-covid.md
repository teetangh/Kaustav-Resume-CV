## Summary

Multi-part project covering COVID-19 data analysis, time-series forecasting, web scraping, and introductory deep learning.

---

## Components

### Data Analysis Report
- Complete data analysis report on worldwide COVID-19 attack statistics
- Used EDA techniques with Matplotlib and Seaborn
- Created barplots, heatmaps, and various statistical visualizations

### Time-Series Forecasting
- Used Facebook's **fbprophet** library for time-series forecasting
- Predicted the number of active corona victim cases in upcoming days

### Choropleth Visualization
- Used Python **folium** package for binding data to maps
- Created choropleth visualizations of COVID-19 data for India

### Web Scraping
- Used **BeautifulSoup** and **Requests** HTTP library
- Scraped live corona statistics from websites

### Neural Networks from Scratch
- Created neural networks from scratch (no ML libraries)
- Implemented a model to recognize the function of an XOR gate without explicit programming

### MNIST Digit Recognition
- Trained a deep learning model using TensorFlow 2 and Keras API
- MNIST handwritten digit recognition task
