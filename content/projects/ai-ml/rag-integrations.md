# Production RAG Integrations

**Type:** Professional project (Couchbase)
**Timeline:** ~2024 -- 2025

---

## Summary

Built production-grade RAG demonstrations integrating Couchbase vector storage with a wide range of AI/ML services and frameworks.

---

## Integrations Built

### Embedding & Reranking Providers
- **Cohere** — embeddings and reranking
- **Voyage AI** — embeddings
- **JinaAI** — embeddings

### LLM Providers
- **Claude AI** (Anthropic)
- **Azure AI** (OpenAI via Azure)
- **AWS Bedrock** (multi-model)

### Agent Frameworks
- **Crew AI** — multi-agent orchestration

### Graph-Augmented Retrieval
- **GraphRAG** — implemented using Microsoft's framework
- Knowledge-graph-augmented retrieval with Couchbase vector storage

### No-Code RAG
- **Langflow-Couchbase** integration
- Enables no-code construction of production RAG pipelines with Couchbase as the vector store
- Reduces time-to-prototype for AI applications

---

## Technologies

- Couchbase Capella (vector database)
- Cohere, Voyage AI, JinaAI
- Claude AI, Azure AI, AWS Bedrock
- Crew AI, GraphRAG, Langflow
- Python
