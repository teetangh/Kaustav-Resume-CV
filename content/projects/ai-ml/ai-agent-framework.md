# AI Agent Tutorial Framework

**Type:** Professional project (Couchbase)
**Timeline:** ~2024 -- 2025

---

## Summary

Tutorial framework enabling enterprise developers to build production-ready agentic AI applications using Couchbase Capella as the vector database backend.

---

## Architecture

- **Capella vector database** — vector storage and retrieval
- **Agent Catalog** — catalog of reusable agent components
- **LangGraph** — agent orchestration and state management
- **LangChain** — LLM application framework
- **LlamaIndex** — data framework for LLM applications

---

## Key Details

- Architected the tutorial framework from the ground up
- Designed for enterprise developers to quickly prototype agentic applications
- Integrates Couchbase's vector search capabilities with popular LLM orchestration frameworks
- Part of Couchbase's AI/ML developer experience strategy

---

## Technologies

- Couchbase Capella (vector database)
- LangGraph, LangChain, LlamaIndex
- Python
